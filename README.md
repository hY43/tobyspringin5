# 토비의 스프링 읽기 모임
* 개요
  * 토비의 스프링을 읽으며, spring 5.3으로 에제를 재작성하는 프로젝트.
  * 각 챕터별 내용과 읽기 모임에 대한 내용 정리는 아래 링크 참고.

* 개발 환경
  * database
    * config : ./docker/mysql/docker-compose.xml
    * exec : docker compose -p springbook up -d
    * ide : mysql workbench (localhost : 3306)

* commit rule
  * feat : 신기능 개발
  * docs : 문서 작성
  * chore : log, comment 등
